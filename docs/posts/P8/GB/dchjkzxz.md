::: note 适用范围

- 本标准规定了电磁环境中控制公众曝露的电场、磁场、电磁场（1Hz~300GHz）的场量限值、评价方法和相关设施（设备）的豁免范围。
- 本标准适用于电磁环境中控制公众曝露的评价和管理。
- 本标准不适用于控制以治疗或诊断为目的所致病人或陪护人员曝露的评价与管理∶不适用于控制无线通信终端、家用电器等对使用者曝露的评价与管理；也不能作为对产生电场、磁场、电磁场设施（设备）的产品质量要求。

:::
