::: note 适用范围

- 本标准规定了水泥工业各生产设备排气筒大气污染物排放限值、作业场所颗粒物无组织排放限值和水泥制品生产的颗粒物排放限值，以及环保相关管理规定等。
- 本标准适用于对现有水泥工业企业及水泥制品生产企业的大气污染物排放管理，以及对新建、改建、扩建水泥矿山、水泥制造和水泥制品生产线的环境影响评价、设计、竣工验收及其建成后的大气污染物排放管理。

:::
